📊  Automobile Sales Analysis Dashboard

## Overview
This Power BI dashboard provides a comprehensive analysis of automobile sales performance, enabling stakeholders to monitor revenue, profitability, regional performance, target achievement, and sales trends through interactive visualisations.

## Dashboard Pages
1. Basic Sales Analysis
   - KPI Cards
   - Sales Trend Analysis
   - Product Performance Analysis
   - Sales vs Profit Correlation

2. Sales – Region Analysis
   - Geographic Sales Distribution
   - Regional Contribution Analysis
   - Waterfall Variance Analysis
   - Regional Performance Comparison

3. Target Sales
   - Sales Funnel
   - Target Achievement Tracking
   - Gauge Performance Monitoring
   - Variance Analysis

4. Catching up with Trends
   - KPI Monitoring
   - Trend Identification
   - Performance Benchmarking
   - Business Growth Insights

## Key Business Insights
- Identifies top-performing products and regions.
- Highlights profit-driving sales segments.
- Tracks target achievement and performance gaps.
- Reveals sales patterns and seasonal trends.
- Supports strategic business decisions using data-driven insights.

## Tools Used
- Power BI
- DAX
- Data Modelling
- Interactive Visualisations
- Business Intelligence Reporting

## Business Value
The dashboard helps decision-makers monitor performance in real time, identify growth opportunities, improve sales strategies, and optimize resource allocation.
